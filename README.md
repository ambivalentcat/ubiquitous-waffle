# ubiquitous-waffle

## GymLingo Pro

Single-page тренажёр английских фраз для зала с:
- аналитикой категорий,
- словарём с фильтрацией и озвучкой,
- режимом флэш-карточек.

## Локальная проверка

Откройте `index.html` (или `GymLingo.html`) в браузере.

## Тесты

```bash
python -m unittest -v tests/test_gym_lingo.py
```
